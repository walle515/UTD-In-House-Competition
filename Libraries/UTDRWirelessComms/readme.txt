UT Dallas Robotics Wireless Communication Library v1
Written by Joey Hines (joey.ahines@gmail.com)

This library allows for parsing packets from serial in the style of 
"~{type}{id}{data}\LF". It is to be used with a ESP8266 and the UTD
Robotics wireless controller.

To Install:
1. Download the UTDRWirelessComms folder
2. Copy folder to the Arduino library folder (Usually located under C:\Program Files\Arduino\Library)
3. Include it in the Arduino Sketch you need it for (#include <UTDRWirelessComms.h>)
4. Compile and Upload Code to Arduino
